# Backlog SAPER / SAPER-Backlog

## 1. Objetivo / Ziel
**PT-BR:** Este backlog organiza as proximas entregas da landing page SAPER. Ele separa o que ja esta concluido, o que deve ser feito agora e o que fica para a proxima fase.

**DE-CH:** Dieses Backlog organisiert die naechsten Lieferungen der SAPER-Landingpage. Es trennt bereits erledigte Punkte, das naechste Was-zu-tun-ist und spaetere Phasen.

## 2. Legenda / Legende
* `done` = concluido / erledigt
* `in_progress` = em andamento / in Bearbeitung
* `pending` = pendente / offen
* `high` = alta prioridade / hohe Prioritaet
* `medium` = media prioridade / mittlere Prioritaet
* `low` = baixa prioridade / niedrige Prioritaet
